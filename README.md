Cordobo Green Park 2
====================

The official WordPress Theme Repository for Cordobo Green Park 2.

**Version 2.0.0** - Modernized for WordPress 6.x

## Overview

Cordobo Green Park 2 is a lightweight, fast-rendering WordPress theme that has been completely modernized to support the latest WordPress features and standards. Originally released during the WordPress 3-4 era with over 100,000 downloads per month, this theme has been revived and updated for modern WordPress installations.

## Features

### Modern WordPress Support
- ✅ **WordPress 6.x Compatible** - Fully tested and optimized for the latest WordPress version
- ✅ **Block Editor (Gutenberg) Support** - Full support for wide/full alignment, block styles, and modern editor features
- ✅ **theme.json** - Modern theme configuration using WordPress theme.json
- ✅ **HTML5 Markup** - Semantic HTML5 structure throughout
- ✅ **Responsive Embeds** - Automatic responsive sizing for embedded content
- ✅ **PHP 7.4+** - Compatible with modern PHP versions

### Security Enhancements
- ✅ **Input Sanitization** - All user inputs properly sanitized
- ✅ **Output Escaping** - All outputs properly escaped to prevent XSS
- ✅ **Nonce Verification** - CSRF protection on all forms
- ✅ **Modern Security Practices** - Following WordPress coding standards

### Design & Layout
- 2-3 column layout with right sidebar
- Fully responsive design
- Custom background support
- Widget-ready (4 widget areas)
- Custom menus (4 menu locations)
- Threaded/nested comments support
- Post thumbnails/featured images
- Custom logo support

### Accessibility
- Accessibility-ready features
- Skip to content links
- Proper ARIA labels
- WCAG compliance improvements

### Translation Ready
- Fully localized
- Multiple language translations included
- Text domain: `greenpark`

## What's New in Version 2.0.0

### Major Updates
- Migrated from beta to stable release

### Breaking Changes

## Installation

1. Download the theme
2. Upload to `/wp-content/themes/` directory
3. Activate through WordPress admin panel
4. Configure via Appearance → Green Park 2 Settings

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher
- Modern web browser (Chrome, Firefox, Safari, Edge)
